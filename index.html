Mastermind Game Rules

Objective

The goal of Mastermind is to guess the secret combination of 4 colored pegs before you run out of attempts or time.

Secret Code

A secret code consisting of four colors is randomly generated at the start of each game.

  1. If "Allow Duplicate" is enabled: Colors in the secret code may repeat (e.g., red, red, yellow, green).
  2. If "Allow Duplicate" is disabled: All four colors in the secret code will be unique.

Attempts

You have a total of 10 attempts to crack the code. The game concludes if you use all 10 attempts without successfully guessing the code.

Feedback (After Each Guess)

After each guess you submit, you'll receive immediate feedback through color-coded dots:

  1. Green: The color is correct, and it's in the correct position
  2. Red: The color is correct, but it's in the wrong position.
  3. Black: The color is not present in the secret code at all.

How to Play

  1. Start the Game: Click the “Play” button to begin.
  2. Enable/Disable Duplicates (Optional): Click “Allow Duplicate” before starting.
  3. Select a Color: Choose any color from the palette.
  4. Place Your Color: Click on a guess slot to place it.
  5. Fill All 4 Slots: Then click the “Check” button.
  6. Check Your Guess: Feedback dots will appear.
  7. Next Turn: Keep guessing until you win or run out of attempts.
  8. Restart the Game: Click “Restart Game” anytime.
